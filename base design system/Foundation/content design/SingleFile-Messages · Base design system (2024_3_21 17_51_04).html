                                                                                          Text messages

                                                                                          Use text messages for marketing purposes and to convey important information regarding topics like schedules, timing, and safety. Texts have the highest open rates and the highest opt-out rates of any form of communication we use, so use them sparingly and purposefully. Send no more than 2 per user per week, unless it’s time-sensitive and transactional.


                                                                                          TL;DR

                                                                                          • Character count is limited to 130 characters in English and 160 characters when translated 

                                                                                          • Put the most important information at the beginning of the text message

                                                                                          • Don't use punctuation unless necessary for comprehension

                                                                                          • Avoid using variables because they can explode character count

                                                                                          • Avoid using abbreviations unless necessary

                                                                                          • Texts can’t have hyperlinked words or shortened links, so use full URLs and make sure to include them in your character count

                                                                                          • Follow these Legal requirements

                                                                                          Anatomy of a text message

                                                                                          • Sender’s name: All texts must start with Uber: so recipients know we’re the sender.

                                                                                          • Text body: Keep it short and meaningful.

                                                                                          • Terms: Include Uber’s terms in the first communication with a user and for each new ride, package delivery, or other service arranged for a third party.

                                                                                          • Unsubscribe: Include unsubscribe instructions in the first communication with a user and for each new ride, package delivery, or other service arranged for a third party. You can use: Text STOP to end msgs.

                                                                                          To see what your text will look like, preview it here.


                                                                                          Writing guidelines

                                                                                          • Make sure text messages add value for the user: We don't want to squander this important communication method.

                                                                                          • Target your texts: They should feel personalized.

                                                                                          • Keep it short: There’s a hard limit of 130 characters in English and 160 characters when translated. 

                                                                                          • Put the most important information first: The critical info or required action should appear at the beginning in case there's truncation.

                                                                                          • Be specific: Prioritize clarity over other considerations. For example, use Your ride was canceled instead of something softer like It looks like your ride was canceled.


                                                                                          Style considerations

                                                                                          • Avoid using variables: If you do use a variable (for example, the recipient’s name), be sure to account for the longest possible version in your character count

                                                                                          • Don’t use punctuation: Only use if it’s necessary for comprehension

                                                                                          • Never use all-caps

                                                                                          • Avoid using abbreviations: Use only when necessary, and then only for familiar abbreviations (for example, msgs for messages)

                                                                                          • Avoid truncating words

                                                                                          • Emoji's are OK: Never use more than one, and keep it at the end of the header or body text, after punctuation

                                                                                          Web links

                                                                                          • Text messages can’t include hyperlinked words: Use full URLs (for example, uber.com/terms) and include them in the character count.

                                                                                          • Don’t add a period after a link inside a text.


                                                                                          Legal requirements

                                                                                          Follow these legal requirements with every text message, without exception:

                                                                                          • Always start with Uber: Let recipients know who’s sending the text.

                                                                                          • Use only a single text for each message: Don't spread the content over several texts.

                                                                                          • Always include the URL for Uber’s terms (uber.com/terms) as well as unsubscribe instructions:

                                                                                            • In the first communication with any user.

                                                                                            • For each new ride, package delivery, or other service arranged for a third party (for example, a ride scheduled by a business for a client).
