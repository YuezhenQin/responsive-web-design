                                                                                                                                Toast

                                                                                                                                A toast is a component for web or tablet used to display short messages or notifications. Toasts often confirm successful or unsuccessful actions, such as submitting a form or completing a task. They're usually displayed for a few seconds before automatically disappearing, but they can remain until dismissed.

                                                                                                                                Usage

                                                                                                                                Use a toast component in a web or tablet environment to provide user feedback after completing an action. It is a nonmodal notification that can alert users of a successful action or inform them of an error.
