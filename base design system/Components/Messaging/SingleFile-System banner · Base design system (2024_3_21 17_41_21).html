                                                                                                                                System banner

                                                                                                                                System banners communicate a global, high-priority state that affects an entire system or app, not just a feature or page. They appear without the user taking an action and follow users from screen to screen. System banners remain until dismissed, or until the state that caused the banner is resolved.
