                                                                                                                                Sliding button

                                                                                                                                Stable

                                                                                                                                Sliding buttons are a variation of our known and loved primary rectangular buttons. Leveraging a different gesture to confirm an action, they help reduce accidental actions and reinforce intent.

                                                                                                                                Screen readers

                                                                                                                                VoiceOver

                                                                                                                                Sliding buttons require a degree of interaction precision, which is not the greatest experience for users using screen readers. For a smoother experience, we recommend using the same interaction a normal button would have.

                                                                                                                                NB: Words like "Slide" should be omitted since swipe gestures do different things when VO is turned on
