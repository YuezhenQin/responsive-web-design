                                                                                                                                Tile

                                                                                                                                A tile is a button-like component that can contain additional controls for selection patterns. The elements inside a tile are stacked in a way that lends themselves to a tile shape which can be square or rectangular, depending on the layout.

                                                                                                                                Metrics

                                                                                                                                Grid spacing mobile
                                                                                                                                Add notes

                                                                                                                                The suggested spacing between tiles on mobile is 8. The suggested spacing around the Tile group is 16. Top or bottom margins may be turned off depending on surrounding content.

                                                                                                                                Grid spacing web
                                                                                                                                Add notes

                                                                                                                                The suggested spacing between tiles on the web is 16 but can be adjusted to match those defined in our layout grids. The suggested spacing around the Tile group is 16 but the spacing can be turned off.
