                                                                                                                                Avatar

                                                                                                                                Stable

                                                                                                                                An avatar is a visual way to represent a person or business in a user interface. In the Uber ecosystem, using avatars encourages empathy and builds trust between earners, merchants, and consumers.

                                                                                                                                Usage
                                                                                                                                Content
                                                                                                                                Spec
                                                                                                                                Status & changelog

                                                                                                                                Metrics, type and color


                                                                                                                                The default avatar starts with blue600 and then alternates in the following order when multiple avatars are present. The color logic is customizable.

                                                                                                                                1. Blue600

                                                                                                                                2. Red600

                                                                                                                                3. Yellow300

                                                                                                                                4. Purple600

                                                                                                                                5. Green600

                                                                                                                                6. Magenta600

                                                                                                                                7. Gray600

                                                                                                                                8. Orange600


                                                                                                                                Screen readers

                                                                                                                                Depending on the context, avatars can have different meanings. They usually need a text alternative unless next to UI text with the same meaning.
