                                                                                                                                Avatar

                                                                                                                                Stable

                                                                                                                                An avatar is a visual way to represent a person or business in a user interface. In the Uber ecosystem, using avatars encourages empathy and builds trust between earners, merchants, and consumers.

                                                                                                                                Usage

                                                                                                                                TL;DR

                                                                                                                                • An avatar can be an icon, a small photo, or the user’s initials

                                                                                                                                • Use an avatar to anchor a user’s profile

                                                                                                                                • In content that accompanies the avatar, include the user’s chosen username or first name only, for privacy
