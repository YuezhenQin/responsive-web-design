                                                                                                                                Page controls

                                                                                                                                Partially stable

                                                                                                                                A page control displays a row of indicators, each of which represents a card, banner, page, or screen in a list.

                                                                                                                                Usage
                                                                                                                                Specs
                                                                                                                                Status & changelog

                                                                                                                                Metrics

                                                                                                                                Spacing between indicators is 8. Top or bottom margins may be adjusted depending on the surrounding content. We recommend a minimum of 8 px margin.
