                                                                                                                                Input and selection

                                                                                                                                Select [Deprecated]

                                                                                                                                Deprecated

                                                                                                                                A select is a type of dropdown menu that allows the user to choose from a list of predefined options. It is typically represented on the page as a dropdown menu when the user clicks on it.

                                                                                                                                Usage
                                                                                                                                Status & changelog
                                                                                                                                Specs

                                                                                                                                Specifics

                                                                                                                                Each out-of-the-box input style has its own nuance. But generally, padding works in 16s and 8s. Icons and certain enhancers are treated centered within a 64pt gutter, to match lists if used full bleed.

                                                                                                                                Mobile tap targets

                                                                                                                                The smallest selects really shouldn’t be used on mobile, but if they have to be, their tap target should be a minimum of 48.
