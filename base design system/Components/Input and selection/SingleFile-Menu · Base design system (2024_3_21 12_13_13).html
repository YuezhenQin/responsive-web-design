                                                                                                                                Menu

                                                                                                                                A Menu is a container to nest various hierarchical information, features, or settings.

                                                                                                                                Usage
                                                                                                                                Status & changelog

                                                                                                                                Anatomy

                                                                                                                                Menu container

                                                                                                                                Menu container anatomy
                                                                                                                                Add notes

                                                                                                                                Each menu is a container with an array of menu items. The container is a temporary surface with 12 round corners and shallow-below drop shadow. The scroll indicator is visible when the list is longer than the container.

                                                                                                                                Menu items

                                                                                                                                Menu items are not just a group of options. Some menus may have sub-menus or multiple selections, and some may need headers and dividers to help organize the options.

                                                                                                                                Search
                                                                                                                                Add notes

                                                                                                                                An extensive menu may require a search field to filter the options.

                                                                                                                                Header
                                                                                                                                Add notes

                                                                                                                                The header is not interactive. It's just a label used to categorize or group menu options.

                                                                                                                                Standard
                                                                                                                                Add notes

                                                                                                                                The standard menu option is just often just a label to name the options. It can have a leading and trailing enhancer icon and a trailing enhancer label. It can also support an optional check box for multiple selections.

                                                                                                                                Chevron
                                                                                                                                Add notes

                                                                                                                                The menu option with a chevron is used to drill down to sub-menus. It can have a leading and trailing enhancer icon and a trailing enhancer label. It can also support an optional check box for multiple selections.

                                                                                                                                Grabber
                                                                                                                                Add notes

                                                                                                                                The menu option with the grabber is highly interactive. Users can re-arrange menu option orders and also pared with an optional switch. It can have a leading and trailing enhancer icon and a trailing enhancer label.

                                                                                                                                Paragraph
                                                                                                                                Add notes

                                                                                                                                The paragraph menu item is not interactive. It is used for instructions or further describing the options.

                                                                                                                                Divider
                                                                                                                                Add notes

                                                                                                                                The divider is a decorative element separating different sections of menu options.



                                                                                                                                Usage 

                                                                                                                                Use a menu to display:

                                                                                                                                • navigational lists

                                                                                                                                • lists of actions



                                                                                                                                Behavior

                                                                                                                                Option menu items are the core elements with different states. The search field in the search menu item has states equivalent to the text field states.


                                                                                                                                States

                                                                                                                                Enabled
                                                                                                                                Add notes

                                                                                                                                The state before a user interacts with an option.

                                                                                                                                Hover
                                                                                                                                Add notes

                                                                                                                                The state that appears when the user pauses over the menu option using a cursor. 4% black

                                                                                                                                Focus
                                                                                                                                Add notes

                                                                                                                                Focus states for menu option show when a user highlights them using a keyboard or voice. 3px border borderAccent.

                                                                                                                                Active
                                                                                                                                Add notes

                                                                                                                                Activate states indicate which option from a group of options is being selected.

                                                                                                                                Disabled
                                                                                                                                Add notes

                                                                                                                                Disabled states indicate when an element isn’t interactive.
