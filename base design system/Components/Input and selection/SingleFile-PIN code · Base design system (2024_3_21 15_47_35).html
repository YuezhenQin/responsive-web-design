                                                                                                                                VoiceOver

                                                                                                                                Attributes anatomy

                                                                                                                                On iOS we use custom attributes in conjunction with some of the standard iOS VoiceOver attributes to provide the same contextual hints on some of the states that a sighted user will see.

                                                                                                                                Most of these values are internal and not something you should worry about when creating VoiceOver specs for screens using any Input component.

                                                                                                                                Attribute name

                                                                                                                                Description

                                                                                                                                inputLabel

                                                                                                                                Reads out the Label element of the field (if one is set)

                                                                                                                                inputValue

                                                                                                                                Reads out the current value of the field

                                                                                                                                inputPlaceholder

                                                                                                                                Reads out the placeholder text of the field

                                                                                                                                inputExplainerLabel

                                                                                                                                Reads out the Hint element of the field (if one is set)

                                                                                                                                accessibilityExplainerPrefixLabel

                                                                                                                                Additional field to read out contextual state information before inputExplainerLabel is read out.

                                                                                                                                accessibilityExplainerSuffixLabel

                                                                                                                                Additional field to read out contextual state information after inputExplainerLabel is read out.

                                                                                                                                accessibilityHint

                                                                                                                                Custom Hint field to be used to override iOS default in some states.

                                                                                                                                length

                                                                                                                                The amount of slots the field has

                                                                                                                                currentSlot*

                                                                                                                                *Internal computed property to indicate which slot the user is currently interacting with.
